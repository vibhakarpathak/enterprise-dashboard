# Enterprise Dashboard (Next.js + React)

## Goal
A scalable, high-performance enterprise dashboard demonstrating
frontend architecture, performance optimization,
and data-driven UI design.

## Tech Stack
- Next.js (App Router)
- React 18
- TypeScript (Strict)
- Redux Toolkit + RTK Query
- GraphQL
- Nx Monorepo

## Architecture
- Monorepo with reusable packages
- Design system driven UI
- Server + Client component split
